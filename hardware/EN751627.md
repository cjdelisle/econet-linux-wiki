# EN751627 xPON/DSL Family
This is a Big Endian MIPS 1004kc based SoC. In DSL applications it is called the EN7516 and in xPON applications it is called the EN7527, though the EN7527 is extremely rare.

EN751627 is *mostly* the same as the EN751221 with a couple of differences.

- Interrupt Controller: EN751627 (as well as EN7528 and EN7580) use a GIC based interrupt controller. Source code for this INTC is here: [kernel49/irq-gic.c](https://github.com/cjdelisle/kernel-49/blob/master/arch/mips/tc3262/irq-gic.c) but it is quite possible that [irq-mips-gic.c](https://github.com/torvalds/linux/blob/master/drivers/irqchip/irq-mips-gic.c) from Linux upstream may work directly out-of-the-box.
- Ethernet: The EN751627 ethernet is *slightly* different from that of the EN751221, the QDMA engines have more interrupts, but they are structurally the same, unlike the EN7580 whose QDMAs are entirely different.
